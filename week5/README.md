# Week 5 Labs

This folder contains lab reproductions completed during **Week 5** of the Data Science and Machine Learning training.

The labs focus on fundamental tools used for **data analysis, numerical computing, and data visualization in Python**.

---

# Week 5 – Day 1  
## NumPy and Pandas

Topics covered:

### NumPy
- Creating NumPy arrays
- Exploring array attributes (shape, size, data type)
- Performing array arithmetic operations
- Understanding vectorized computations

### Pandas
- Creating Pandas Series
- Creating Pandas DataFrames
- Accessing elements within Series and DataFrames
- Selecting specific columns and rows
- Combining DataFrames using concatenation

---

# Week 5 – Day 2  
## Introduction to Matplotlib

Topics covered:

- Introduction to data visualization using Matplotlib
- Importing the Matplotlib library
- Creating basic plots using `matplotlib.pyplot`
- Understanding figure and axis concepts
- Creating simple charts for data representation
- Visualizing datasets for better data interpretation

Matplotlib is one of the most widely used Python libraries for **data visualization in Data Science and Machine Learning**.

---

# Contents of Each Lab

Each lab folder includes:

- Jupyter notebooks containing the executed code
- Step-by-step documentation explaining the process
- Screenshots showing the results of the executed commands
- Observations and lessons learned during the lab

These labs help reinforce practical skills in **data manipulation, numerical analysis, and visualization using Python**.
